Link Preview Project

Introduction

Project is a web service that extracts preview data e.g. title, description, image from a website. These entities are stored in the database for later listing of added webpages. Later, unneeded entries can be deleted.

Technologies

Backend: Node.js, Express framework.

Frontend: JavaScript, jQuery, AJAX, Bootstrap.

Database: MongoDB (hosted by mLab).
